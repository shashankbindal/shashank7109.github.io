Portfolio — Shashank Bindal

A high-performance, aesthetically driven portfolio built with Next.js, Tailwind CSS, and Framer Motion.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Features
- **Premium Design**: Dark mode first, glassmorphism, and custom noise textures.
- **Dynamic Interactions**: Custom 3D tilt effects, typewriter animations, and smooth scroll reveals.
- **SEO Optimized**: Full metadata suite, robots.txt, and sitemap generation.
- **AI-ML Showcase**: Detailed sections for production RAG pipelines and AI schedulers.

## Tech Stack
- **Frontend**: Next.js 15, React 19, Tailwind CSS, Framer Motion.
- **Icons**: Lucide React, Heroicons.
- **Fonts**: Manrope, Playfair Display, DM Sans.

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:4028](http://localhost:4028) to view the result.
